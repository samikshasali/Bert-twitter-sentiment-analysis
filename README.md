# Bert-twitter-sentiment-analysis
# 🚀 BERT Fine-Tuning for Twitter Sentiment Analysis

##  Overview
Fine-tuned a pre-trained BERT model to classify tweets into sentiment categories (positive, negative, neutral).

---

##  Tech Stack
- Python  
- PyTorch  
- Hugging Face Transformers  

---

##  Workflow
- Data preprocessing (cleaning text, handling missing values)  
- Tokenization using `bert-base-uncased`  
- Fine-tuning BERT model  
- Evaluation using Accuracy, Precision, Recall, F1 Score  

---

##  Results

| Metric     | Score |
|-----------|------|
| Accuracy  | 51%  |
| Precision | 45%  |
| Recall    | 51%  |
| F1 Score  | 46%  |

---

##  Experiments
- Frozen BERT layers  
- Fine-tuned last layers (better performance)  

##  Conclusion
BERT improves text classification performance, especially with fine-tuning.
